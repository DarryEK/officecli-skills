# OfficeCLI 图表配色指南 & Excel/WPS 专业图表配色库

OfficeCLI xlsx chart `colors` 属性用法指南 + 复刻 [chart-color-palettes](https://github.com/kailes/chart-color-palettes) 开源库的 27 套精选色板。

## 内容

- `SKILL.md` — 完整技能文件，包含 OfficeCLI colors 用法 + 27套色板 + 20种图表样式指南
- `README.md` — 本文件

## 色板一览

| 分类 | 套数 | 适用场景 |
|------|------|----------|
| 基础必备 | 5 | 通用报表、学术、风险 |
| 商务职场 | 5 | 财务、述职、政企、制造 |
| 莫兰迪进阶 | 5 | 人事、极简、零售、医疗、教育 |
| 工业/数据统计 | 5 | 运维、机械、销售、环保 |
| 学术/马卡龙/预警 | 5 | 论文、绩效、新媒体、IT |
| 暗色大屏 | 2 | 数据大屏、监控仪表盘 |

## 使用方法

在 officecli 命令中 chart 的 `--prop colors="..."` 属性使用（去掉 `#` 前缀，逗号分隔）。
