# OfficeCLI Thesis Workbench

本科毕业论文标准化交付工作台。以标准和证据为骨架，四种 DOCX 交付路径为输出。

## 适合

- 有真实项目需要写成毕业论文的学生
- 需要严格匹配学校模板格式的用户
- 导师批注后需要系统化修订的用户
- 素材不足、需要联网检索或头脑风暴的用户

## 四种交付路径

- **M1** 默认样式生成 — Markdown → DOCX
- **M2** 样文版式贴近 — Markdown + 样文分析 → 样式合并 DOCX
- **M3** 模板副本填充 — Markdown + 学校模板 → 模板格式不变 DOCX（推荐）
- **M4** 已有稿编辑 — 初稿 DOCX + 编辑指令 → 文本级修改

完整说明见 [SKILL.md](SKILL.md)。
